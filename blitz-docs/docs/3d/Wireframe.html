Wireframe

Created on 2023-04-25 19:23 Never Updated

This command enables or disables wireframe rendering.

This will make each polygon only render as an outline, without any shaded areas.
Very useful for debugging purposes.

It is disabled by default on all video modes.

Parameters

Parameters marked with an asterisk (*) are optional.

  1. Enable% - True to enable wireframe rendering, and False to disable it.
