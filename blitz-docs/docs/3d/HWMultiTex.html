HWMultiTex

Created on 2023-04-26 19:03 Never Updated

This command enables or disables hardware multi-texturing.
Multi-texturing is a technique used to display multiple textures in a single object.

It is enabled by default in all modes.

Remarks All modern GPUs support hardware multi-texturing. Using this command is kind of useless.

Parameters

Parameters marked with an asterisk (*) are optional.

  1. Enable% - True to enable hardware multi-texturing, and False to disable it.
