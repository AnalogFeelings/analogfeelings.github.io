Graphics3D

Created on 2023-04-21 20:20 Never Updated

Sets 3D Graphics mode. This command must be executed before any other 3D command, otherwise programs will return an error.

Width and height set the resolution of the game window. The resolution must be compatible with the GPU and monitor being used.

Depth sets the colour mode of the screen. If this value is omitted or set to 0, then the highest available colour depth available will be used.
It is recommended to use 32-bit for better Windows 8+ compatibility.

Parameters

Parameters marked with an asterisk (*) are optional.
  1. Width% - The window width in pixels.
  2. Height% - The window height in pixels.
  3. Depth%* - The window color depth in bits.
  4. Mode%* - Mode of display.
    • 0 - Windowed in debug mode, fullscreen if in release config.
    • 1 - Fullscreen.
    • 2 - Windowed.
    • 3 - Windowed or scaled.
