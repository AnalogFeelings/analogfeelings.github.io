AmbientLight

Created on 2023-04-26 19:14 Never Updated

This command sets the ambient light color.

Ambient light is a light source that affects all objects equally, which provides no shading.
It is useful to add some level of light before adding other light sources for realism.

An ambient light of RGB(0, 0, 0) will result in no ambient light.
The default ambient light value is RGB(127, 127, 127).

Parameters

Parameters marked with an asterisk (*) are optional.

  1. Red# - The red color channel value.
  2. Green# - The green color channel value.
  3. Blue# - The blue color channel value.
