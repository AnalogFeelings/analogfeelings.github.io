AntiAlias

Created on 2023-04-25 19:04 Never Updated

This command enables or disables anti-aliasing.

Anti-aliasing is a technique used to smooth out jagged edges to improve the visuals of the game.
It is disabled by default in all video modes.

Remarks Due to a DirectX 7 limitation, this command will only have an effect in Fullscreen modes.

Parameters

Parameters marked with an asterisk (*) are optional.

  1. Enable% - True to enable anti-aliasing, and False to disable it.
