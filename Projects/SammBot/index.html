Features

⚙

Administration

Allows you to manage the bot through commands!
Including restarting, shutting down, saying messages, and more!

❔

Help

I think this is pretty self-explanatory.
It just lets you get a list of modules and commands!

🎲

Fun

Hug users, pat them, ask the magic 8-ball, and more!

ℹ

Information

Get information from the bot, users, or the current guild!

🧑‍⚖️

Moderation

Allows moderators to kick, ban, mute and
even bulk-delete messages!

👥

Profiles

Per-user profiles!
Currently only supports pronouns. The
bot will use these pronouns in some commands.

🎰

Random

Randomized crazyness!
Get a cat, dog, duck or even fox image! Hell, you can even get a random SCP article.

🏷️

User Tags

Allows users to create tags!
A tag is composed of a name and a reply. When a user gets this tag, Samm-Bot replies with the set reply!

🔧

Utilities

Get user avatars, visualize color codes, or even get your city's weather forecast!
