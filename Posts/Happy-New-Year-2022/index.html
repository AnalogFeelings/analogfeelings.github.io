Annoying Cookies Prompt

Heya!

My website uses a single cookie (well, actually two since I need to keep track of this dialogue box) for themeing!

If you consent, I will only use that cookie to keep track of the website's look (Windows 98 or XP theme).
If you don't consent instead, the website will always display the Windows 98 theme (or maybe Windows XP if the prompt cookie expired before the theme cookie).

Error!

You need to consent to cookies before you can change your theme!
You will be prompted for cookie consent every 2 months.

As a workaround, go here, follow the guide and edit the "AcceptedCookies" value to "Yes".
